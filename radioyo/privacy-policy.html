PRIVACY POLICY FOR RADIOYO APPLICATION

Based on a templated provided by https://www.docracy.com/6016/mobile-privacy-policy#

Hereafter, the application known as Radioyo will be referred to as The Application.

What information does the Application obtain and how is it used?

The Application does not obtain any personal user information

User Provided Information

The Application does not obtain any user information!

Automatically Collected Information

The Application does not collect any personal user information! It does not collect usage statistics, location information, bug information, users' phone usage, external files or personal details. The Application does allow the user to specify custom radio station streaming links and associated data, but this information is not uploaded to any server.

Do third parties see and/or have access to information obtained by the Application?

When the user accesses internet radio streams using The Application, their device's IP address and MAC are sent so the third party servers, so that those servers know where to send the requested data to. Outside of this, third parties have no access to any personal information obtained by The Application because no such information is obtained! Radio stations saved by the user are saved to the device only. No usage statistics are gathered or transmitted by the application. Such information may or may not be gathered by servers hosting radio stations that are linked to by The Application, but this is unknown to the Application, and its creator.

What are my opt-out rights?

You are free to install and uninstall The Application, and no personal data will be requested, obtained, or otherwise collected by The Application before, during or after this time.

Data Retention Policy, Managing Your Information

The only data retained by The Application is a list of internet radio station urls, and associated metadata, that the user is free to add to, and/or amend. This information is not attached to any user account or any name. This list of internet radio station as stored by The Application is not transmitted by The Application to any external server, and are removed on deletion of The Application.
